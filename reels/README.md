## ⚡ Comment les enregistrer en mp4 (3 minutes)

### Sur Mac
1. Ouvre le fichier `.html` dans **Chrome ou Safari** (double-clic).
2. Passe le navigateur en **plein écran** : `⌃⌘F` (Control + Cmd + F).
3. Lance le screen recording : **`⌘⇧5`** (Cmd + Shift + 5).
4. Sélectionne **"Enregistrer une portion sélectionnée"** et cadre la zone du Reel (la portion verticale au centre).
5. Clique sur le bouton **"↻ Rejouer"** en haut à droite du Reel, puis sur **"Enregistrer"** dans la barre de capture.
6. Laisse jouer 15 à 18 secondes selon le Reel, puis arrête (`⌘⌃Esc` ou clic stop dans la barre de menu).
7. Le mp4 atterrit sur le bureau.

### Sur PC Windows
1. Ouvre le fichier `.html` dans Chrome (double-clic).
2. Plein écran : `F11`.
3. Lance la **Xbox Game Bar** : `⊞ + G` (Touche Windows + G).
4. Clique sur **"Capture"** → bouton rond rouge pour démarrer.
5. Clique **↻ Rejouer** dans le Reel.
6. Arrête le recording après 15-18 sec.

## 🎵 Ajouter une musique au moment de publier

Au moment de poster sur Instagram :
1. Ouvre Instagram → bouton **"+"** → **Reel**.
2. Importe le mp4 que tu viens d'enregistrer.
3. Clique sur **"Musique"** (icône note de musique).
4. Choisis selon le Reel :
   - **Reel 1 (Afroboost)** : un afrobeat moderne avec un beat clair (Burna Boy, Rema, Tems, ou tape "afrobeat" dans la recherche).
   - **Reel 2 (Silent Cardio)** : chill electronique avec drop vers la fin (Bonobo, Ben Böhmer style, ou cherche "sunset chill").
   - **Reel 3 (La rencontre)** : bossa-house ou afrobeat romantique sans niaiserie.

## 🔧 Tu veux modifier un Reel ?

Les fichiers `.html` sont 100% modifiables. Tu peux :
- Changer le texte d'un Reel : ouvre le fichier dans un éditeur de texte, cherche le texte à changer (entre `>` et `<`), modifie.
- Ajuster les couleurs : tout est dans la section `:root` au début du `<style>`.
- Modifier les durées : chaque "stage" (étape) a une animation avec `Xs forwards` que tu peux ajuster.

Si tu veux qu'un graphiste les retravaille dans After Effects pour les améliorer, donne-lui le fichier `.html` et les screenshots des captures — ça lui servira de base de référence visuelle exacte.
