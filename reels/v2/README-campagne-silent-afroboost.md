## 🎙️ Comment utiliser les profils du site comme matière première

Tu m'as dit que tu vas souvent utiliser les **profils du site en mode aperçu** pour faire des Reels avec des accroches vocales. Voici le workflow optimal :

### Étape 1 — Capture un profil
- Va sur ton site → connecte-toi → ouvre le profil de quelqu'un (toi-même, un coach, un membre) → clic sur "Aperçu de ton profil".
- Screen recording de cette page mobile (`⌘⇧5`).

### Étape 2 — Customise le template `00`
Ouvre `00-template-profil-apercu.html` dans un éditeur de texte. Cherche les zones marquées `👉 EDIT` :
- **Avatar** : remplace par `<img src="URL_PHOTO" style="width:100%;height:100%;object-fit:cover;border-radius:50%">`
- **Prénom + ville** : modifie le texte
- **Bio** : copie-colle la bio du profil
- **Badges sports** : ajuste les sports listés
- **Hook (Stage 1)** : adapte la phrase d'accroche selon ton message

### Étape 3 — Enregistre ta voix par-dessus
- Soit dans Instagram directement (option "Voix off" lors du montage Reel).
- Soit en amont avec ton téléphone (Mémo vocal) → tu importes dans ton Reel.
- Le template a déjà une **barre voice prompt animée** qui simule visuellement ton audio. Synchronise ta voix au démarrage de l'animation.

### Étape 4 — Publie
Screen recording du HTML → mp4 → Instagram → ajoute musique d'ambiance → légende + hashtags → publie.
