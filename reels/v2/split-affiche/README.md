# 🎬 3 Reels "Split" d'affiche Silent Afroboost

3 Reels Instagram qui, vus ensemble, forment ton affiche horizontale complète. Technique de **storytelling cohérent** très efficace : 1 affiche = 3 publications, chacune avec son angle.

## 📂 Fichiers
- `split-1-gauche.html` — Hook "Pourquoi le Silent ?"
- `split-2-centre.html` — Localisation + ambiance "Au bord du lac de Neuchâtel"
- `split-3-droite.html` — Offre découverte + CTA réservation (le plus important)
- `affiche-source.jpg` — **À AJOUTER PAR TOI** (ton image originale)

## ⚡ Comment ça marche en 4 étapes

### 1. Place ton image
Mets ton affiche dans ce dossier sous le nom exact :
```
affiche-source.jpg
```
(ou `.png` — les fichiers HTML détectent les 2)

### 2. Vérifie l'aperçu
Ouvre chaque fichier HTML dans Chrome. Tu dois voir :
- **Split 1** : la partie GAUCHE de ton affiche en grand (Bassi + logo Afroboost + "POURQUOI L'EXPÉRIENCE SILENT ?")
- **Split 2** : la partie CENTRE (Bassi qui danse + groupe en arrière-plan)
- **Split 3** : la partie DROITE (le groupe + le coucher de soleil)

### 3. Enregistre chaque Reel (15 sec chacun)
Pour chacun des 3 :
1. Plein écran : `⌃⌘F` (Mac)
2. `⌘⇧5` → "Enregistrer une portion sélectionnée"
3. Cadre la zone verticale du Reel
4. Clique **↻ Rejouer** dans le Reel pour relancer l'animation
5. Démarre l'enregistrement → 15 secondes → stop

Tu obtiens 3 fichiers .mp4 sur ton bureau.

### 4. Publie sur Instagram dans l'ordre
**Ordre strict** : Split 1 → Split 2 → Split 3.

Espace les publications de **5 minutes** entre chaque. Pourquoi ?
- Ça permet à Instagram de bien indexer chaque Reel.
- Sur ta grille de profil, les 3 vont s'aligner et former l'affiche complète.
- Les viewers qui swipent vont vivre une vraie mini-narration.

## 💡 Tips pour booster l'engagement

1. **Stories en parallèle** : 5 min avant chaque Reel, fais une story qui dit "Nouveau Reel dans 5 min, surveille bien".
2. **Question dans le 2e Reel** : sous le post, écris dans les commentaires une question : *"T'as déjà testé le Silent Disco ?"* → relance les conversations.
3. **Repost du 3e** : 24h plus tard, repost UNIQUEMENT le 3e Reel (celui avec l'offre) en story avec sticker "Premier cours offert ⚡".
4. **Hashtag de marque** : crée `#SilentAfroboostNeuchâtel` et utilise-le sur les 3 Reels — c'est le pivot de ta campagne.
